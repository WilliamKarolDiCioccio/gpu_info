# gpu_info

GPU info is a simple and fast Dart plugin designed to retrieve information about the user's machine GPU using a Vulkan graphics API instance. It is being developed as part of [OpenLocalUI](https://github.com/WilliamKarolDiCioccio/open_local_ui). The plugin is currently in its early stages and has not yet been published.

## Roadmap

Below is a set of basic features that the plugin will offer:

1. **Retrieve GPU information:** Allow users to obtain basic information about their GPU, such as model, vendor, and memory.

3. **Cross-platform support:** Ensure compatibility across multiple platforms, including Windows, macOS, and Linux.

Once these features are implemented, the plugin will be ready for initial release. Additionally, ongoing development will focus on enhancing functionality and improving overall developer experience.
